At this point I feel like I've narrowed it down to some problem with my Card class. I googled some stuff and learned about a class method called 'unicode'. I added it to my Card class:

class Card(object):
    '''Generates a card with a suit, rank'''

    ##use class to create unicode representation

    def __init__(self, suit, rank):
        self.suit = suit
        self.rank = rank

    def __str__(self):
        return unicode(self).encode('utf-8')

    def __unicode__(self):
        return '%s of %s' % (self.rank, self.suit)
